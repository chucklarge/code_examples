Lines Simulation

or Lines Screensaver, whichever you prefer

Requirements:

Python >= 2.5
Pygame >= 1.7.1

That's it!

On Windows, if you want to use lines.exe as a screen saver, you have to use exe2scr, which is available freely online (google it).

The main program is lines.py, lines.exe, or lines.app
The settings module is settings.py, settings.exe, or settings.app
